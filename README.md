# Florian Portfolio - Site Web Portfolio 3D

Portfolio moderne pour un artiste 3D, construit avec React, Three.js (react-three-fiber) et Tailwind CSS.

## Technologies

- **React** - Framework JavaScript
- **Vite** - Build tool et dev server
- **React Three Fiber** - Bibliothèque React pour Three.js
- **Tailwind CSS** - Framework CSS utilitaire
- **React Router** - Routing côté client

## Installation

```bash
npm install
```

## Développement

```bash
npm run dev
```

## Build

```bash
npm run build
```

## Structure du projet

```
src/
├── components/     # Composants réutilisables
├── pages/         # Pages du site
├── assets/        # Assets (images, modèles 3D)
├── styles/        # Styles globaux
└── App.jsx        # Composant principal
```
